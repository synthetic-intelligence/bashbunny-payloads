# Registry Uploader

Author: CoulsTech
Version: 1.0

## Description

Sets a single preconfigured registry key by compounding the individual settings into a PowerShell command.
Intended for administrative use, or whatever purpose suits your "update registry key quickly and easily" needs.

## Configuration

Registry key data and path configuration in the payload.txt file
(Needs to point at an existing registry value & path that you intend to change)

## Requirements

Windows 10/11
